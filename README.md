# Universal-Turing-Machine
Universal Turing Machine that can simulate the functionality of any Turing Machine made using C++

<br>t 1 Z 2 1 H
<br>t 2 1 2 0 L
<br>t 2 0 2 1 L
<br>t 2 Z 2 Z H
<br>f 2
<br>i 00000000
<br>i 00000001
<br>i 01100110
